QUICK REFERENCE
===============

▶ First Time Setup
------------------
1. pip install requests beautifulsoup4
2. Edit config.json with your artists
3. python venue_scraper.py


▶ Daily Use
-----------
python venue_scraper.py              # Run scraper
python venue_scraper.py --export     # Run and export to JSON
python test_demo.py                  # Test with mock data


▶ Automation
------------
python run_scheduler.py              # Run on schedule (edit schedule first)

Or use cron:
0 2 * * * cd /path/to/venue-scraper && python venue_scraper.py


▶ View Results
--------------
Python:
  from database import VenueDatabase
  db = VenueDatabase()
  print(db.get_all_venues())

SQLite:
  sqlite3 venues.db "SELECT * FROM venues"

JSON:
  python venue_scraper.py --export
  cat venues_export.json


HOW IT WORKS
============

1. Reads config.json for artist list
2. Queries Bandsintown & Songkick APIs
3. Extracts venue information
4. Compares against database (fuzzy matching)
5. Adds new venues only
6. Logs everything to venue_scraper.log


CUSTOMIZATION
=============

Add more artists:
  Edit config.json

Add more scrapers:
  Create new scraper in scrapers/
  See README.md "Extending the Scraper"

Change database location:
  python venue_scraper.py --database /path/to/db

Change schedule:
  Edit run_scheduler.py schedule settings


TROUBLESHOOTING
===============

No events found:
  • Check artist name matches exactly
  • Some artists may not have upcoming shows
  • Check venue_scraper.log for details

Duplicates appearing:
  • Adjust similarity threshold in venue_manager.py
  • Default is 0.85 (85% match required)

Need help:
  • Check venue_scraper.log
  • Run python test_demo.py
  • See README.md for full docs
